## Title: Password Manager Program
## Description: This program allows a user to secure  their passwords for 
free in an offline database The program comes with multiple features 
that allows them to generate passwords, test the strength of them, and 
check if they have ever been leaked. This allows the user to 
forget about passwords and copy them to their clipboard when needed.
## Features:
Password manager
Password generator
Copy to clipboard
Password strength testing tool
Breach Scanner
Major Skills:
OOP, JComponents
Control statements
,Timers
,APIs

## APIs used
JFreeChart used for the ring chart
HaveIbeenpwned API for the breach scanner
https://haveibeenpwned.com/API/v3#SearchingPwnedPasswordsByRange
Passay API for password generation
https://mvnrepository.com/artifact/org.passay/passay
Openkeepass APi for Password Manager
https://mvnrepository.com/artifact/de.slackspace/openkeepass/0.5.0
Extra libraries needed for the password manager to work
https://mvnrepository.com/artifact/org.simpleframework/simple-xml/2.7.1
https://mvnrepository.com/artifact/com.madgag.spongycastle/core/1.58.0.0
Nbvcxz API for Password strength test
https://github.com/GoSimpleLLC/nbvcxz
